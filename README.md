 Hendelek Assistant Bot

Telegram bot powered by Groq AI (LLaMA 3)

## What it does
- Answers any questions via Telegram
- Powered by Groq API with LLaMA 3 model
- Fast responses

## Tech Stack
- Python
- Groq API
- python-telegram-bot

## How to run
1. Clone the repo
2. Install dependencies: `pip install python-telegram-bot groq python-dotenv`
3. Create `.env` file with your keys
4. Run: `python bot.py`
